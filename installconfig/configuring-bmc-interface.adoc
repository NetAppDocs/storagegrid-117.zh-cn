= BMC界面：概述(SGF6112、SG6000、SG100和SG1000)
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
SGF6112、SG6000或服务设备上的底板管理控制器(BMC)用户界面可提供有关硬件的状态信息、并可用于配置设备的SNMP设置和其他选项。

在安装设备时、请使用本节中的以下过程配置BMC：

* 更改 BMC 接口的根密码
* 设置 BMC 管理端口的 IP 地址
* 访问 BMC 界面
* 配置SNMP设置
* 设置电子邮件通知和警报



NOTE: 如果此设备已安装到网格中且正在运行StorageGRID 软件、请参见以下过程、了解BMC配置的替代机制。
